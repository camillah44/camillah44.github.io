FLOODNET
FLOODNET was a java applet created by Electronic Disturbance Theater in 1998.
Through FLOODNET, users could send messages to a web server's error logs, in an effort to make a permanent statement and even slow down the server.
FLOODNET used a unique loophole in HTML to create a functional art piece - voicing the concerns of the people with functionality.
This was the first instance of "Electronic Civil Disobediance."
